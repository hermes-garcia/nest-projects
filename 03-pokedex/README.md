# Pokedex Project
<hr>

## Development
1. Clone this repository
2. Run:
```
yarn install
```
3. Install Nest CLI
```
npm i -g @nest/cli
```
4. Compose database
```
dpcker-compose up -d
```

## Stack
* NestJS
* MongoDB
